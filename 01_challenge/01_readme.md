# Challenge 01

Функция `fizzbuzz(max_num)` отвечает за вывод чисел по правилам FizzBuzz: для кратных 3 печатает `fizz`, для кратных 5 печатает `buzz`, а для кратных 3 и 5 одновременно печатает `fizzbuzz`.

Ошибка: в условии `elif` использовался оператор присваивания `=` вместо сравнения `==`.
Решение: заменил `=` на `==`, после чего проверка кратности 3 начала работать корректно.
