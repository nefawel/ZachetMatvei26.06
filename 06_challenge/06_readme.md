# Challenge 06

Функция `fizzbuzz(max_num)` отвечает за использование значений из файла `conf.py` и вывод FizzBuzz с этими настройками.

Ошибка: модуль конфигурации читался по несуществующему атрибуту `conf.num`.
Решение: заменил обращение на `conf.num2`, который действительно объявлен в `conf.py`.
