# Challenge 03

Функция `fizzbuzz(max_num)` отвечает за обработку числового аргумента и вывод строк `fizz`, `buzz` или `fizzbuzz` для подходящих чисел.

Ошибка: в `fizzbuzz` передавалась строка вместо целого числа, из-за чего `range()` выбрасывал `TypeError`.
Решение: заменил строковый аргумент на число `100`, чтобы программа формировала полный корректный вывод.
