# Challenge 04

Функция `fizzbuzz(max_num)` отвечает за проверку кратности чисел 3 и 5 и вывод соответствующего текста по правилам FizzBuzz.

Ошибка: строка `print` после `if` имела неправильный отступ, поэтому возникал `IndentationError`.
Решение: исправил отступ и вернул корректную структуру блока условия.
