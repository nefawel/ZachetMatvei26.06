# Challenge 07

Функция `fizzbuzz(max_num)` отвечает за вывод всех значений FizzBuzz в заданном диапазоне, включая последнее нужное число.

Ошибка: программа запускалась с аргументом `99`, а цикл `range(1, max_num)` не включает верхнюю границу.
Решение: изменил вызов на `fizzbuzz(100)`, чтобы вывод включал число 99.
