# Challenge 05

Функция `fizzbuzz(max_num)` отвечает за чтение входных данных из файла `myfile.txt` и дальнейший вывод результата FizzBuzz по этим значениям.

Ошибка: файл открывался по неверному имени, использовался синтаксис `print` из Python 2, а диапазон не включал 99.
Решение: подключил правильный файл `myfile.txt`, убрал проблемный вывод и сделал чтение/цикл корректными для полного FizzBuzz до 99.
