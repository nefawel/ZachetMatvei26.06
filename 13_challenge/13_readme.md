# Challenge 13

Функция `fizzbuzz(max_num)` отвечает за полный вывод FizzBuzz и должна корректно запускаться из блока `if __name__ == '__main__':`.

Ошибка: в файле было сразу несколько синтаксических и логических проблем: незакрытая строка, пропущенное двоеточие в `for` и неверное имя функции в вызове.
Решение: закрыл строковый литерал, добавил двоеточие и заменил `fizzbuzzy(100)` на `fizzbuzz(100)`.
