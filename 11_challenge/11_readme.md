# Challenge 11

Функция `fizzbuzz(max_num)` отвечает за правильный порядок проверок, чтобы программа отдельно обрабатывала `fizz`, `buzz` и `fizzbuzz`.

Ошибка: проверка на кратность 3 выполнялась раньше проверки на кратность и 3, и 5 одновременно, поэтому `fizzbuzz` никогда не печатался.
Решение: переставил условия в правильном порядке и привёл `buzz` к нижнему регистру.
