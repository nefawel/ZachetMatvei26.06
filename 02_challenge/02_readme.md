# Challenge 02

Функция `fizzbuzz(max_num)` отвечает за перебор чисел в диапазоне и вывод результата FizzBuzz для кратных 3 и 5.

Ошибка: функция `fizzbuzz` вызывалась без обязательного аргумента `max_num`.
Решение: передал корректное значение `100` и заодно привёл вывод `buzz` к ожидаемому виду в нижнем регистре.
